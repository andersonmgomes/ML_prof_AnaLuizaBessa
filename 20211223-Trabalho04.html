2. Modelos Preditivos¶

A seguir ser realizada uma breve contextualizao sobre os modelos preditivos selecionados para este trabalho.

## 2.1 KNN

O algoritmo K-nearest neighbors (K-NN) um mtodo no paramtrico de classificao que pode ser utilizado tanto para problemas de regresso quanto para problemas de classificao.

A ideia central do algoritmo utilizar-se do clculo da distncia da observao cuja classe desconhecida em relao aos demais, cujas classes j so conhecidas. A distncia pode ser calculada por diversos mtodos, como o Euclidiano, o de Hamming, o de Manhattan e o de Markowski.

No K-NN a estrutura do modelo ser determinada pelo dataset utilizado. Este algoritmo tambm conhecido como de aprendizado lento ou melhor dizendo, um algoritmo preguioso (lazy). Os algoritmos do tipo lazy, no necessitam de dados de treinamento para se gerar o modelo, o que diminui em partes o processo inicial, mas em contrapartida gerar uma necessidade de analise posterior mais apurada.

## 2.2 Naive Bayes

O algoritmo Naive Bayes um classificador probabilstico utilizado em tarefas de aprendizado supervisionado, cujo funcionamento se baseia no Teoremo de Bayes.

A ideia central do mtodo considerar, conforme Teorema de Bayes, que as features so independentes entre si, ou seja, que a presena de uma no afeta quaisquer uma das outras. A partir dessa premissa, o algoritmo calcula a probabilidade da observao pertencer a uma determinada classe a partir dos dados disponveis nas variveis independentes.

Seu principal ponto forte a capacidade de performar bem com um pequeno conjunto de dados de treinamento, principalmente quando as variveis independentes so categricas. Por outro lado, para grande parte das aplicaes reais, a premissa de independncia entre as variveis no se confirma, o que afeta o desempenho do mtodo.

## 2.3 rvore de Deciso

O algoritmo rvore de Deciso utiliza modelos de rvore para tomada de deciso em problemas de classificao. Nele, as folhas representam as classes e os galhos representam um determinado conjunto de testes sobre os valores das features.

As rvores de Deciso exigem menos esforo na fase de preparao de dados. Desnormalizao e ausncia de valores em determinadas observaes, por exemplo, no afetam o modelo. tambm um mtodo fcil e intuitivo, o que permite um bom nvel de "explicabilidade".

Por outro lado, pequenas alteraes nos dados podem afetar decisivamente o modelo. Frequentemente a fase de treinamento tem um alto custo computacional e no pode ser aplicada para problemas de regresso.

## 2.4 MLP

Multilayer Perceptron (MLP) um tipo de rede neural artificial simples, composto por trs camadas de ns: uma camada de entrada (input layer), uma camada oculta (hidden layer) e uma camada de sada (output layer).

O MLP utiliza uma tcnica de aprendizado supervisionado chamada backpropagation na fase de treinamento. O treinamento ento ocorre quando o perceptron muda seus pesos a partir da anlise do erro de sada em comparao ao resultado esperado.

O mtodo MLP pode ser aplicado para resolver complexos problemas no lineares. Funciona bem com grandes conjuntos de dados, ao mesmo tempo que consegue boa acurcia com menores datasets. Sua principal desvantagem o alto custo computacional na fase de treinamento.

## 2.5 SVM

O algoritmo Support Vector Machines (SVM) um mtodo de aprendizado supervisionado que se utiliza da otimizao de hiperplanos de um espao n-dimensional (n igual ao nmero de features) para separar as distintas classes do conjunto de dados.

Para tal, utiliza-se do clculo de distncia entre os pontos para definir a fronteira de deciso. Por meio de uma tcnica chamada Kernel Trick, o mtodo capaz de aumentar artificialmente a dimensionalidade dos dados, a fim de obter a melhor separao entre as classes.

Seu principal ponto forte funcionar bem quando o nmero de dimenses maior do que o nmero de exemplos. Por outro lado, tecnicamente invivel para grande conjunto de dados, devido aos custos computacionais envolvidos.
